# 📸SNS

게시물을 작성하고 타임라인과 피드에서 게시물을 구경합니다.

## About the SNS project

평상시에 많이 사용하는 Instagram 어플의 주요 기능을 담은 웹페이지를 만들어보았습니다.

## Features

### Main Features

* 회원가입 및 로그인 기능

* 게시물 작성 기능

* 타임라인 기능
  * 좋아요, 댓글 쓰기

* 페이지 권한 부여 기
  * 비로그인시, 접근 불가
